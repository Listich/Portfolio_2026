# Portfolio — Serena Listich

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Déploiement GitHub Pages

```bash
npm run deploy
```

> Penser à adapter `base` dans `vite.config.js` selon le nom du repo :
> - repo `listich.github.io` → `base: '/'`
> - repo `portfolio` → `base: '/portfolio/'`

## Photo de profil

Placer la photo dans `src/assets/profile.jpg`
